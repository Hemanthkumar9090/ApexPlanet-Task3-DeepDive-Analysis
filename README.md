# Task 3 - Deep Dive Analysis & Interactive Dashboard

## 📌 Objective
The objective of this project is to perform a deep business analysis on the Superstore dataset by identifying key performance indicators (KPIs), discovering sales trends, analyzing customer behavior, and generating business insights using Python and interactive visualizations.

## 📂 Dataset
- Dataset Name: Superstore.csv
- Records: Superstore Sales Data
- Source: ApexPlanet Internship

## 🎯 Core KPIs
- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Profit Margin

## 📊 Analysis Performed
- Monthly Sales Trend Analysis
- Sales by Category
- Sales by Region
- Top 10 Products Analysis
- Profit by Segment
- Sales vs Profit Analysis
- Correlation Heatmap
- Customer Segmentation using K-Means Clustering

## 💡 Business Insights
- Identified top-performing product categories.
- Compared regional sales performance.
- Analyzed customer segments using K-Means clustering.
- Evaluated the relationship between sales and profit.
- Studied the impact of discounts on profitability.
- Generated business recommendations for better decision-making.
